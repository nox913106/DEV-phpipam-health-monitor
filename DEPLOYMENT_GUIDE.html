phpIPAM Health Dashboard

Deployment Guide v2.2.1

New Installation

Step 1: Prerequisites

  • Docker phpIPAM v1.7.4 running
  • MariaDB container accessible
  • Git installed on host

Step 2: Clone Repository

cd /tmp
git clone https://github.com/nox913106/DEV-phpipam-health-monitor.git
cd DEV-phpipam-health-monitor

Step 3: Database Setup

# Copy SQL file to container
docker cp database/health_check_tables.sql phpipam_phpipam-mariadb_1:/tmp/

# Execute SQL
docker exec phpipam_phpipam-mariadb_1 mariadb -u phpipam -p phpipam < /tmp/health_check_tables.sql

Step 4: File Deployment

# Copy health dashboard files
docker cp dashboard/. phpipam_phpipam-web_1:/phpipam/health_dashboard/
docker cp api/. phpipam_phpipam-web_1:/phpipam/health_dashboard/api/
docker cp includes/. phpipam_phpipam-web_1:/phpipam/health_dashboard/includes/

# Set permissions
docker exec phpipam_phpipam-web_1 chown -R www-data:www-data /phpipam/health_dashboard/

Step 5: Setup DHCP Monitor (v2.2.1)

# Create directory structure on host
mkdir -p /opt/health_check/scripts /opt/health_check/config

# Copy files
cp scripts/dhcp_monitor_daemon.php /opt/health_check/scripts/
cp config/dhcp_servers.json /opt/health_check/config/

# Add volume mount to docker-compose.yml for phpipam-cron:
# volumes:
#   - /opt/health_check:/health_check

# Add command to docker-compose.yml for phpipam-cron:
# command: >
#   sh -c "
#     if [ -f /health_check/scripts/dhcp_monitor_daemon.php ]; then
#       nohup php /health_check/scripts/dhcp_monitor_daemon.php >> /var/log/dhcp_monitor.log 2>&1 &
#     fi
#     exec /sbin/tini -- /bin/sh -c 'crond -f'
#   "

# Restart container
docker-compose up -d phpipam-cron

Step 6: Verification

# Check Dashboard
curl -I https://YOUR_SERVER/health_dashboard/

# Check DHCP daemon
docker exec phpipam_phpipam-cron_1 ps aux | grep dhcp_monitor

# Check database
docker exec phpipam_phpipam-mariadb_1 mariadb -u phpipam -p phpipam -e \
  "SELECT * FROM health_check_dhcp_history ORDER BY recorded_at DESC LIMIT 10;"

API Documentation

Get Latest Status

GET /health_dashboard/api/api_stats.php?action=latest

Get System History

# Fixed period (hours: 1, 3, 6, 8, 12, 24)
GET /health_dashboard/api/api_stats.php?action=system_history&hours=6

# Custom time range
GET /health_dashboard/api/api_stats.php?action=system_history&start_time=2025-12-22%2000:00&end_time=2025-12-22%2005:00

Get DHCP History

GET /health_dashboard/api/api_stats.php?action=dhcp_history&hours=3

Get DHCP Server List

GET /health_dashboard/api/api_dhcp_config.php?action=list

MCP Tools

# Daily Health Check
GET /api/mcp/tools/daily_health_check/?dhcp_server_ip=172.16.5.196

# Weekly Health Report
GET /api/mcp/tools/weekly_health_report/
