phpIPAM Health Check Manual

Complete Documentation for Health Monitoring System

Overview

The phpIPAM Health Check System provides real-time monitoring of system resources and DHCP servers.

Version v2.2.1
System Monitoring Every 5 minutes (CPU, Memory, Disk)
DHCP Monitoring Every 5 seconds
Data Retention 7 days (auto cleanup)

Dashboard Access

Standalone URL https://ipam-tw.pouchen.com/health_dashboard/
Embedded URL https://ipam-tw.pouchen.com/index.php?page=tools§ion=health-monitor

Features

  • Real-time system resource monitoring (CPU, Memory, Disk)
  • DHCP server availability monitoring with 5-second intervals
  • 24-hour historical statistics with charts
  • Light/Dark mode theme support
  • Multi-language support (EN/ZH-TW/ZH-CN)
  • DHCP server management UI
  • Flexible time range queries

API Endpoints

# Get latest status
GET /health_dashboard/api/api_stats.php?action=latest

# Get system history
GET /health_dashboard/api/api_stats.php?action=system_history&hours=6

# Get DHCP history  
GET /health_dashboard/api/api_stats.php?action=dhcp_history&hours=3

# Get DHCP server list
GET /health_dashboard/api/api_dhcp_config.php?action=list
